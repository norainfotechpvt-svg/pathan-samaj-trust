# પઠાન સમાજ ચેરિટેબલ ટ્રસ્ટ (Pathan Samaj Charitable Trust)

મુસ્લિમ સ્વામિનારાયણ એકતા સમાજ માટેનું એક ડિજિટલ રજિસ્ટ્રેશન અને ફંડ મેનેજમેન્ટ પ્લેટફોર્મ.

## સુવિધાઓ (Features)
- **સભ્ય રજિસ્ટ્રેશન:** ફોટો અને પર્સનલ વિગતો સાથે સુરક્ષિત નોંધણી.
- **પઠાન દીકરી યોજના:** ૧ કરોડની સહાય યોજના માટેનું ખાસ ફોર્મ.
- **ફંડ મેનેજમેન્ટ:** દૈનિક, માસિક અને સામાન્ય ફાળાનો હિસાબ.
- **વ્યક્તિગત પાસબુક:** દરેક સભ્ય પોતાનો જમા-ઉપાડનો હિસાબ જોઈ શકે છે.
- **ગુજરાતી સપોર્ટ:** રકમ અને વિગતો ગુજરાતી શબ્દોમાં જોવાની સુવિધા.
- **ડેટા સિક્યોરિટી:** બધો ડેટા યુઝરના બ્રાઉઝરમાં સુરક્ષિત રીતે સેવ રહે છે.

## GitHub પર લાઈવ કેવી રીતે કરવું (How to make it Live)
૧. બધી ફાઇલો GitHub પર અપલોડ કરો.
૨. **Settings** ટેબમાં જાઓ.
૩. ડાબી બાજુ **Pages** પર ક્લિક કરો.
૪. 'Build and deployment' માં **main** બ્રાન્ચ સિલેક્ટ કરી **Save** આપો.
૫. ૧-૨ મિનિટમાં ઉપર એક લિંક આવશે, તેના પર ક્લિક કરવાથી તમારી વેબસાઇટ દુનિયાભરમાં ચાલુ થઈ જશે.

## ટેકનોલોજી (Technology Stack)
- React.js (Frontend Framework)
- Tailwind CSS (Modern UI Design)
- TypeScript (Strongly Typed Code)
- LocalStorage (Offline Data Storage)
